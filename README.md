# beGitter
## サイト概要
### サイトテーマ
プログラミング初心者の為のSNS

### テーマを選んだ理由
プログラミング学習を始めた当初、ひとりでの学習に不安や難しさを感じ、モチベーションが下がることもありました。  
しかし、Xで他の学習者のデイリーレポートを見ることで励まされ、学習への意欲が高まりました。  
同じように努力する仲間の存在が支えとなったことから、学習の進捗を共有し、互いにモチベーションを高め合えるSNSアプリを作りたいと考え、このテーマを選びました。​

### ターゲットユーザ
- プログラミング初心者の人
- 同じ境遇で仲間が欲しい人
​
### 主な利用シーン
- 一日の学習が終了したとき
- 比較対象が欲しい時
- 詰まってモチベーションが下がってきたとき
